## project-dashboard.html - hover 提示公式修正

重写了 `getProjectTooltip()` 中的状态计算逻辑：

**进行中项目：**
- 已延期天数 = max(0, 今日日期 - 项目结束日期)
- 可能延期天数 = max(0, (实际开始日期 - 项目开始日期) - 已延期天数)
- 提示：可能延期>0且已延期≤0 → "可能延期xx天"；两者都>0 → "已延期xx天，还可能延期xx天"；仅已延期>0 → "已延期xx天"

**已完成项目：**
- 不再使用 todayTime，改用 completedDate 与 projectEnd 对比
- 延期完成天数 = completedDate - projectEnd（>0延期，=0按时，<0提前）
