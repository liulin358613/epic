# 2026-05-22

## project-dashboard.html 甘特图修复

### 今日日期同步
- 3处硬编码 `new Date(2026, 4, 21)` 全部改为 `new Date()`（系统当前日期）
- 涉及：进度计算（1154行）、今日日期列高亮（1253行）、tooltip（1340行）

### 蓝色框（今日日期列高亮）
- 最终方案：`position: fixed` + 视口坐标（`getBoundingClientRect`）
- 挂载到 `document.body`，监听 scroll/resize 动态更新位置
- 高度 = container 的可视区域高度，上下各超出 6px

### 表头固定
- `overflow-y: auto; flex: 1; min-height: 0` 垂直滚动
- `border-collapse: separate; border-spacing: 0` 防透视
- `th { position: sticky; top: 0; z-index: 10 }` + box-shadow 补边框
- `.th.date-header` 去掉 `position: relative`（否则覆盖 sticky）

### 响应式缩放
- JS: `PROJECT_COL_WIDTH/NAV_COL_WIDTH/DATE_CELL_WIDTH` 从模块顶层常量改为函数内动态计算（`SCALE = ROOT_FS / 16`）
- CSS: `px` → `rem`（padding、列宽、导航列宽）
- `style.width` 从 px 改为 rem

### today 归一化到零点
- `getDateStatus`(1154行) 和 `getProjectTooltip`(1341行) 的 `new Date()` 后加 `today.setHours(0,0,0,0)`
- 原因：项目日期都是零点，today 带时分秒时 `Math.ceil` 算天数差会在凌晨多算一天

### hover 效果限定为日期列
- `bindProjectHover()` 从绑定整行 `tr` 改为只绑定 `td.date-cell`
- 删除 `#gantt-table-body tr:hover` 整行 hover 背景色
- 删除 `.nav-btn:hover` 背景色，仅保留文字变色
- `bindProjectHover()` 调用移入 `renderGanttTable()` 末尾，确保每次重绘后重新绑定

### 顶部预警统计卡片同步项目数据
- 4个统计卡片的 `.stat-value` 添加 `data-stat` 属性（delayed/warning/completed/delayed-done）
- 新增 `updateStatsCards()` 函数，从 `projectData` 动态计算各状态数量
- 统计逻辑：已延期=未完成且today>endDate，可能延期=delayDays>0或isDelayed，按时完成=completed:'onTime'/'early'，延期完成=completed:'delayed'
