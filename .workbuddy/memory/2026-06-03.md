**前两版失败原因：** 纯 CSS 方案无法精确处理 flex gap 下相邻子列间横线连接。

**第三版（JS+CSS变量）：**

**修复：**
- 静态 HTML（resetView）：改为嵌套结构
- 动态 HTML（updateOrgChart）：新增 buildMemberNode() 和 buildOrgBranch() 辅助函数
