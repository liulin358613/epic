# 2026-05-28 工作日志

## create-org.html 页面布局响应式适配

调整了 C/create-org.html 的布局排版，使其适配所有浏览器和屏幕分辨率：
- 新增大屏（≥1920px）font-size 17px 适配
- 所有 `@media` 改为 `@media screen` 写法，兼容性更好
- `header-content` 宽度改为 `min(75rem, 92%)`，避免极宽屏溢出或极窄屏截断
- `register-body` 加 `padding: ... 1rem` 左右边距，防止贴边
- `register-box` 的 `max-width` 从 `90%` 改为 `100%`（body 已有左右 padding，无需再限宽）
- `≤600px` 断点补充 `.register-body` padding 调整、`.form-label` 宽度缩小
- `≤400px` 新增极小屏断点
- 其他任何内容（颜色、字体、功能、文本）均未修改
